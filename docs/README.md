### 3. [prototype.html](./prototype.html)
**交互式HTML原型**，可直接在浏览器中打开查看和体验：

#### 功能特性
✅ **完全可交互**：所有核心功能均可操作
- 添加任务（快速添加到"现在"或"未来"）
- 拖拽任务在三个视图间移动
- 编辑任务（标题、截止日期、备注）
- 完成任务（移至历史）
- 删除任务
- 搜索/过滤任务

✅ **真实数据**：包含示例数据
- 现在：3个任务（含逾期、今天、未来截止日）
- 未来：2个任务
- 历史：2个已完成任务

✅ **完整UI反馈**
- Toast通知（含撤销功能）
- 拖拽高亮效果
- 容量警告（现在 > 5个任务）
- 空状态提示

✅ **键盘快捷键**
- `N`：聚焦快速添加
- `1/2/3`：切换视图
- `/`：聚焦搜索
- `Enter`：添加任务（Shift+Enter添加到"现在"）

✅ **响应式设计**
- 支持桌面端和移动端
- 自适应布局

✅ **本地存储**
- 使用LocalStorage持久化
- 刷新页面数据不丢失

#### 如何使用
1. 直接在浏览器中打开 `prototype.html`
2. 体验添加、拖拽、编辑等操作
3. 尝试键盘快捷键
4. 刷新页面查看数据持久化效果

## 💡 设计理念

本项目遵循**极简主义**设计哲学：

- **三个核心视图**：现在、未来、历史
- **零学习成本**：直观的拖拽交互
- **聚焦当下**：现在视图建议≤5个任务
- **快速收集**：一键添加，减少阻力
- **轻量反馈**：Toast通知，无阻塞弹窗
- **撤销机制**：降低操作焦虑

**不做什么**（保持简洁）：
- ❌ 标签/项目/优先级矩阵
- ❌ 子任务/依赖关系
- ❌ 复杂统计图表
- ❌ 多人协作功能
