## 📊 执行总结

### 项目概况
Todo Flow 是一个极简主义的任务管理应用，采用"现在 / 未来 / 历史"三栏视图设计，帮助用户专注于当下任务，合理规划未来工作，并回顾历史完成情况。

## 📚 使用说明

### 快速启动

#### 1. 启动服务
```bash
# Windows
start-dev.bat

# Linux/Mac
./build.sh
```

#### 2. 访问应用
```
http://localhost:5173
```

#### 3. 测试账号
无需登录，直接使用

### 基本操作

#### 添加任务
1. 点击右下角浮动按钮（+）
2. 输入标题（必填）
3. 可选：添加备注、截止日期
4. 选择添加到"现在"或"未来"
5. 点击"创建任务"

#### 完成任务
1. 在任务卡片上点击"完成"按钮
2. 任务自动移动到"历史"

#### 编辑任务
1. 点击任务标题
2. 在详情抽屉中修改
3. 点击"保存"

#### 搜索任务
1. 在顶部搜索框输入关键词
2. 实时过滤当前视图

#### 筛选任务
1. 使用快速筛选按钮
2. 或使用排序下拉框

### 快捷键
- `N` - 添加到未来
- `Shift+N` - 添加到现在
- `1` - 切换到"现在"
- `2` - 切换到"未来"
- `3` - 切换到"历史"
