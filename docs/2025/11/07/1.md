## 🧩 Todo Flow 首页任务列表优化方案（融合你的想法）

### 一、任务列表布局与文字层级

1. **文字大小与层级调整**

   * 标题（任务名）可提升至 `16px–18px`，字重 `600`，增强视觉抓点。
   * 描述信息保持 `13–14px`，灰度略浅，区分主次。
   * 状态/时间等辅助信息用 `12px`，半透明灰白。

2. **内容与按钮的对齐方式**
   当前“文字在左、按钮在右，中间空”导致视觉重心偏两端。
   ✅ 改进方案：

   * 采用“分层布局”：

     ```
     [任务标题 + 状态标签] 
     [描述 + 日期 + 操作按钮(并列)]
     ```
   * 或者在卡片右下角集中动作按钮，让中间区域填充描述和时间，整体更协调。

---

### 二、任务操作功能拓展

1. **新增编辑按钮**

   * 放在“完成”按钮旁，采用笔形图标 ✏️，点击后弹出任务编辑弹窗。
   * 支持快速修改标题、描述、日期、优先级等。

2. **扩展操作集（除完成/删除外）**

   | 动作    | 功能描述                | 交互建议         |
   | ----- | ------------------- | ------------ |
   | 🕓 延期 | 修改任务日期为次日/自定义日期     | 点击后弹出日期选择器   |
   | 🔁 移动 | 在「现在 / 未来 / 历史」之间切换 | 下拉菜单或拖拽操作    |
   | ⭐ 置顶  | 将任务固定到当前列表顶部        | 图标固定变亮       |
   | 🧩 编辑 | 修改任务内容、标签、备注        | 弹出编辑窗口       |
   | 🗑 删除 | 二次确认后删除             | 保留 hover 才显示 |

---

### 三、任务创建逻辑优化

1. **默认日期自动设为当天**

   * 新建任务时自动带上当前日期。
   * 若用户切换到“未来”页创建任务，则默认日期为次日。
   * 提供清除按钮支持“无日期”选项。

2. **可选：快捷创建面板增强**

   * 支持键入 `/today`、`/future`、`/nextweek` 自动分配日期。
   * 在输入框下方展示创建预览（例如：“将在 11月8日 添加到 当前任务”）。

---

### 四、交互与视觉细节

* “完成”按钮变为绿色主色调；“编辑/删除”次级灰。
* 操作按钮可组合成一个「…」菜单，减少视觉噪音。
* 任务 hover 时轻微放大或高亮，反馈更自然。
* 切换“现在/未来/历史”时，使用淡入淡出或滑动过渡。
