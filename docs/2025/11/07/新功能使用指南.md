## 🚀 快速开始

本次更新为任务列表带来了更丰富的交互功能和更优雅的视觉体验。

## ✨ 新增功能一览

### 1. ✏️ 快速编辑按钮

**位置**：每个任务卡片右侧，绿色"完成"按钮旁边

**外观**：铅笔图标 ✏️

**使用方法**：
1. 找到要编辑的任务
2. 点击任务右侧的铅笔图标
3. 在弹出的编辑窗口中修改内容
4. 点击"保存"完成编辑

**适用场景**：
- 快速修改任务标题
- 更新任务描述
- 调整截止日期

**提示**：无需打开任务详情抽屉，直接点击编辑按钮即可！

---

### 2. ⋯ 更多操作菜单

**位置**：编辑按钮右侧

**外观**：三个点 ⋯

**使用方法**：
1. 点击三点图标打开菜单
2. 选择需要的操作
3. 菜单自动关闭并执行操作

**包含功能**：

#### 🕓 延期
- **功能**：自动将任务截止日期改为明天
- **适用场景**：今天做不完，推迟到明天
- **操作**：点击"延期" → 日期自动更新为明天
- **提示**：会显示"任务已延期至明天"的通知

#### 🔁 移动
- **功能**：在"现在"和"未来"之间切换
- **适用场景**：调整任务优先级
- **操作**：
  - "现在"的任务 → 移动到"未来"
  - "未来"的任务 → 移动到"现在"
- **提示**：会显示"任务已移至未来/现在"的通知

#### ⭐ 置顶（即将推出）
- **功能**：将任务固定到列表顶部
- **当前状态**：功能预留，完整实现需要后端支持
- **计划**：下一版本完成

#### 🗑 删除
- **功能**：永久删除任务
- **操作**：点击"删除" → 确认 → 任务被删除
- **警告**：删除操作不可逆，请谨慎使用
- **替代方案**：如果只是完成任务，请使用"完成"按钮

---

### 3. 📅 智能默认日期

**功能**：创建新任务时自动设置合理的默认日期

**规则**：
| 创建位置 | 默认日期 | 说明 |
|----------|---------|------|
| "现在"页面 | 今天 | 立即开始的任务，今天完成 |
| "未来"页面 | 明天 | 稍后处理的任务，明天或之后 |

**使用方法**：
1. 点击 `+` 按钮或按快捷键 `N`（未来）/`Shift+N`（现在）
2. 输入任务标题和描述
3. 日期字段已自动填写，无需手动选择
4. 如需修改，点击日期选择器或使用快捷按钮
5. 如需清除日期，点击日期右侧的 `✕` 按钮

**快捷按钮**：
- **今天**：设置为当天
- **明天**：设置为次日
- **下周**：设置为 7 天后

**提示**：系统会根据你创建任务的位置智能推荐日期，节省时间！

---

### 4. 🎨 视觉与动画优化

#### 任务卡片
- **标题更大**：16px，字重 600，更突出
- **描述清晰**：13px，灰色，最多显示 2 行
- **布局优化**：分两行显示，视觉更平衡
- **Hover 效果**：鼠标悬停时卡片轻微放大，阴影加深

#### 按钮样式
- **完成按钮**：绿色主色调，表示积极操作
- **编辑按钮**：灰色，hover 变蓝色
- **删除按钮**：红色，表示警告操作

#### 页面切换
- **淡入淡出**：切换"现在/未来/历史"时，页面平滑过渡
- **滑动效果**：从右侧滑入，向左侧滑出，方向感明确

#### 历史记录
- **半透明显示**：已完成的任务显示为 60% 不透明度
- **完成时间**：显示"完成于 X天前"（中文）

## 🎮 操作演示

### 场景 1：快速延期今天的任务

```
今天做不完某个任务，想推迟到明天：

1. 在"现在"页面找到该任务
2. 点击任务右侧的三点图标 ⋯
3. 选择"🕓 延期"
4. 任务日期自动更新为明天
5. 收到"任务已延期至明天"的通知

✅ 完成！只需 2 次点击。
```

### 场景 2：将未来任务移到现在

```
未来任务变得紧急，需要立即开始：

1. 在"未来"页面找到该任务
2. 点击任务右侧的三点图标 ⋯
3. 选择"🔁 移动"
4. 任务自动移到"现在"列表
5. 收到"任务已移至现在"的通知

✅ 完成！只需 2 次点击。
```

### 场景 3：快速编辑任务

```
需要修改任务标题或描述：

1. 找到要编辑的任务
2. 点击铅笔图标 ✏️
3. 在弹出窗口中修改内容
4. 点击"保存"

✅ 完成！无需打开详情抽屉。
```

### 场景 4：创建新任务（自动日期）

```
在"现在"页面创建新任务：

1. 点击右下角 `+` 按钮或按 `Shift+N`
2. 输入标题："完成项目报告"

4. 日期已自动设为今天（无需手动选择！）

✅ 完成！日期自动设置，节省一步操作。
```

## 💡 使用技巧

### 技巧 1：批量延期
如果多个任务都需要延期，可以逐个点击"延期"按钮，比打开编辑窗口逐个修改日期快得多。

### 技巧 2：快速调整优先级
使用"移动"功能在"现在"和"未来"之间快速切换，无需手动修改状态。

### 技巧 3：利用智能日期
- 紧急任务在"现在"页面创建，系统自动设为今天
- 计划任务在"未来"页面创建，系统自动设为明天
- 创建后可随时修改日期

### 技巧 4：减少视觉干扰
次要操作（延期、移动、删除）都隐藏在三点菜单中，界面保持简洁。

### 技巧 5：使用 Hover 预览
鼠标悬停在任务卡片上，卡片会轻微放大并高亮，帮助你快速定位。

## 🔔 注意事项

### 关于延期功能
- 延期只改变截止日期，不改变任务状态
- 如果任务没有日期，延期会设置日期为明天
- 连续延期会累加（第二次延期变成后天）

### 关于移动功能
- "历史"任务不显示移动按钮（已完成的任务不能移动）
- 移动操作会立即生效，无需额外确认
- 移动后任务会出现在目标列表中

### 关于删除功能
- 删除操作不可逆，请谨慎使用
- 如果任务已完成，建议使用"完成"按钮而不是删除
- 删除会弹出确认对话框，防止误操作

### 关于编辑功能
- 编辑模态框支持 `Esc` 键关闭
- 修改后点击"保存"才会生效
- 如果不想保存，点击"取消"或直接关闭窗口

## 📱 移动端使用

所有新功能在移动端同样可用：

- **编辑按钮**：点击铅笔图标
- **三点菜单**：点击三点图标打开
- **菜单关闭**：点击菜单外部任意位置
- **Hover 效果**：移动端不显示 hover，但保留点击反馈

**提示**：移动端按钮区域经过优化，点击区域足够大（44x44 像素），避免误触。

## 🐛 常见问题

### Q: 为什么历史任务没有编辑和三点菜单？
**A**: 历史任务是已完成的任务，原则上不应再修改。如需修改，可以重新创建一个新任务。

### Q: 延期功能可以选择具体日期吗？
**A**: 延期功能默认改为明天。如需选择具体日期，请使用编辑按钮打开编辑窗口。

### Q: 置顶功能什么时候可以用？
**A**: 置顶功能已预留接口，完整实现需要后端支持，预计下一版本完成。

### Q: 菜单打开后怎么关闭？
**A**: 点击菜单外部任意位置，或选择任意菜单项后自动关闭。

### Q: 创建任务时可以不设置日期吗？
**A**: 可以。虽然系统会自动设置默认日期，但你可以点击日期右侧的 `✕` 按钮清除日期。
